# AI-Ковчег — Кейсы и результаты 1 потока

> Social proof для продажника. Каждый кейс — живой человек из 1 потока с нишей, запросом и результатом. Упоминать в переписках для убеждения «это работает и для твоей ниши».

## Флагманские кейсы (топ для переписок)

### Кейс 1. «Знакомый Антона» — уволил 200 человек
- **Ниша:** бизнес на 300 сотрудников (детали не раскрываются, упоминается на Воркшопе №4)
- **Запрос:** операционная нагрузка, ФОТ растёт, ощущение что всё на себе
- **Что сделал:** за месяц внедрил несколько AI-ассистентов в маркетинг и продажи
- **Результат:** уволил 200 человек из 300. Ему не нужна была такая команда
- **Где упомянуть:** при разговоре с владельцем бизнеса с командой, про «Завод в кармане»
- **Стори вышли в Instagram Влада и Антона**

### Кейс 2. Татьяна Хавро — психотерапевт
- **Ниша:** психотерапия, распаковка эксперта
- **Запрос:** упаковать свою экспертизу, выйти из индивидуальной работы
- **Что сделала:** загрузила материалы в NotebookLM → обратный инжиниринг методологии → запустила **бот персонализированных медитаций**
- **Инструменты из курса:** NotebookLM + Claude Project + вайб-кодинг для бота
- **Где упомянуть:** для психологов, коучей, эзотериков, ведущих трансформационных игр

### Кейс 3. София Дробинина — ассистент продаж в переписках
- **Ниша:** эксперт по продажам в переписках, наставник
- **Запрос:** вытащить свою методологию из головы, упаковать продукт
- **Что сделали:** 10 часов её контента → 80 скриншотов разборов → извлекли методологию → создали AI-ассистента → приложение-CRM
- **Срок:** ключевой ассистент сделан за **4 часа вайб-кодинга**
- **Инструменты:** Claude Code + NotebookLM + GitHub → Railway
- **Стал стандартом кейса:** обратный инжиниринг экспертизы → скилл Sofia, который используется внутри команды для обработки лидов
- **Где упомянуть:** любому эксперту, который хочет масштабироваться через AI

### Кейс 4. Jedi Monkey — Дофаминомика
- **Ниша:** блогер с СДВГ, любопытный энтузиаст
- **Запрос:** от праздного любопытства к продукту
- **Что сделал:** цепочка «любопытство → Deep Research → база знаний в NotebookLM → Claude Code → продукт „Симулятор дофамина“»
- **Результат:** **2288 пользователей с одного поста без рекламы**
- **Где упомянуть:** творческим людям, блогерам, тем кто «хочу сделать свой продукт, но не знаю какой»

### Кейс 5. Запуск на $120 000 с помощью AI
- **Ниша:** (детали в кейс-материалах, используется как цифра)
- **Результат:** запуск с $120K выручки, собранный через стек AI-инструментов курса
- **Где упомянуть:** цифра для якорения в продающих сообщениях

## Антикейсы / честность

Из ToV Влада: не прячет провалы и сырость продукта. «Уязвимость = доверие».

- Программа тестовая → могут быть косяки → компенсируем гиперценностью
- Иван-куратор **не спит** — жизнь разделяется на «до» и «после» (это честный disclaimer)
- Группа 3-5 человек — не все доходят до результата. 50/50 ответственность ученика и тренера
- «Это не волшебная таблетка, нужно работать» (Антон на Воркшопе №1)
- «Не ждите, что за 1.5 месяца вы станете теми, кто в мире зарабатывает миллиарды, используя AI»

Эта честность — конкурентное преимущество vs инфоцыгане.
