# 🚢 kovcheg-sales-agent

> **Claude Code Skill** — AI-ассистент менеджера по продажам курса AI-Ковчег

8-недельная программа AI-автоматизации для предпринимателей. Тарифы: $450 / $1 000 / $3 500.

Этот скилл воплощает **методологию КВП + Софии Добрыниной** + два голоса менеджеров (Анастасия эмпатичная / Владислав деловой), собранные из реальных Zoom-звонков и 4 успешных Telegram-сделок.

## ⚡ Установка

### Через git clone (рекомендуется)

```bash
cd ~/.claude/skills

```

### Перезапуск Claude Code

После клонирования перезапусти Claude Code (или открой новую сессию).

### Проверка

В Claude Code напиши:
```

```

или просто:
```
Помоги ответить лиду AI-Ковчега
```

— скилл должен подцепиться автоматически.

## 🚀 Как использовать

### Сценарий 1 — Ответить клиенту в Telegram

```
[Вставляешь сообщение клиента из ТГ]
```

Скилл проведёт диагностику (язык, темперамент, архетип, этап КВП-скрипта), выберет голос (Анастасия/Владислав) и предложит ответ в правильной структуре.

### Сценарий 2 — Анализ переписки

```
Проанализируй эту переписку и предложи следующее сообщение:
[вставляешь весь диалог]
```

### Сценарий 3 — Работа с возражением

```
Клиент написал «Дорого, я подумаю». Что ответить?
```

### Сценарий 4 — Закрытие на оплату

```
Клиент готов оплатить. Какой следующий шаг?
```

## 🎯 На чём работает

- **Методология КВП** — внутренний скрипт отдела продаж AI-Ковчега (5 принципов + 8 блоков разговора)
- **Методология С. Добрыниной** — SPIN, температура лида, формула Цель → Вопросы → Решение → Действие
- **2 реальных голоса** — извлечены из 3 Zoom-звонков и 4 ТГ-переписок реальных менеджеров
- **15+ реальных кейсов** учеников 1-го потока для аргументации в нише клиента

## 🔄 Связь с другими скиллами

- `sales-assistant` — общая методология Добрыниной (без привязки к продукту)
- `kovcheg-sales-agent` (этот) — методология **+ продуктовая база AI-Ковчега**

Если у тебя другой продукт — используй `sales-assistant`.
Если AI-Ковчег — используй этот.

## 💡 Философия

> **Этот скилл — НЕ замена менеджеру.** Это усилитель.
>
> Менеджер закрывает 1 сделку за 6 дней (87 сообщений в среднем).
> Скилл может вести 50 диалогов параллельно, не теряя качества.
>
> Когда нужна душа — менеджер. Когда рутина — скилл.

Полностью совпадает с философией всего AI-Ковчега: **«Делегируй 80% рутины нейронкам».**
